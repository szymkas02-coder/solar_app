# SolarScope — PV Calculator

A single-page solar energy calculator built with pure HTML, CSS, and JavaScript. No build step or backend required — open  in any browser.

## Features

- Interactive Leaflet.js map for selecting any location
- PV system sizing: panel count, tilt angle, azimuth, system losses
- Annual and monthly energy yield estimates using real irradiance data (PVGIS API)
- Financial analysis: investment cost, payback period, 25-year savings projection
- Dark-themed UI with Plotly charts and a monospace/modern font pair

## Tech Stack

Vanilla JS · Leaflet.js · Plotly.js · PVGIS API (European Commission)

## Usage

```bash
# No installation needed
open index.html
```

Or serve it with any static file server:

```bash
python -m http.server
```
